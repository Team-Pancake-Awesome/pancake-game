# pancake-game
Game about pancakes cooking themselves

## Controls (keyboard/mouse)
Mouse up and down controls the spatula's tilt.
A/D is left/right.
Tapping space is scoops a pancake.
Holding space while a pancake is scooped allows you to flip it by flicking the mouse up.
Tapping space while a pancake is scooped allows you to set the pancake down.
Pressing enter while a pancake is scooped sends the pancake to the selected customer.
Right/left arrow keys are back/forward in the order queue.
Up/down arrows keys control the stove.

You can interact with the topping dispensers (the colored cubes) by pressing down on them with the spatula.
